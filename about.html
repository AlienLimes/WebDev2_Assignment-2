Welcome Page

This page asks for your favorite color and gives you a greeting.

  • Changes the page title with your name
  • Special colours are ( RED, GREEN, BLUE)

Data Manipulation Page

This page has a colour Theme Selector function that will create a colour theme.

  • User has to select exactly 4 colours, not three or five.
  • When the four colours selected, press Show colours
  • If the user wants another colour theme, just de-select the colours

About Page

About page tells the user what functions are available and plus has an extra color information search database.

  • Enter a colour
  • Then press search to see more information about that colour
  • Available colours are :
    -Black
    -Green
    -Red
