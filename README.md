# Website Development Assignment 2

Interactive JavaScript website.

## Description
User can create a personal colour template by selecting four colours.
Dynamic writing to updating webpage content.
CSS animation, transition
CSS Framework Tailwind
Handlebar templating to display data

## Features

1. **Feature 1**: Image flip, Image before/ after .
2. **Feature 2**: Colours database with CRUD functions.
3. **Feature 3**: Change background on about page.
